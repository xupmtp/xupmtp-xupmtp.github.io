使用GCP建立Valheim Server

步驟說明

  1. 在Google Cloud Platform(GCP)上建立VM
  2. 安裝Valheim Server
  3. 搬移舊地圖

建立VM

1. 啟動GCP服務
  1. 進入Google Cloud頁面
  2. 點選免費試用並輸入信用卡資訊(3個月試用期，結束可解除綁定)
  3. 新增專案
2. 建立vm
  1. 點選左邊工具列 -> Compute Engine -> VM執行個體 -> 建立
  2. 填入圖中圈起部分，其中藍色須按圖中資料填寫，紅色可根據自身需求修改，完成後按建立便完成
    • 左邊區域可依自身所在區域設定，右邊區域選擇任一選項皆可，沒特別影響
    • 核心部分最低2vCPU/4GB，提供3~5人遊戲已十分夠用，若進入遊戲內覺得卡頓可再調高(若調高須注意流量是否夠用)
    • 變更按鈕調整作業系統設定，防火牆未勾選會讓其他玩家無法連線
    • 點選變更後按下圖輸入，100G以valheim來說已經十分夠用
3. 修改對外IP
  1. 點選左邊工具列 -> VPC網路 -> 外部IP位址
  2. 選擇類型，修改臨時->靜態(使用者必須是你的vm名稱) -> 名稱自訂 -> 確定
4. 修改防火牆
  1. 點選左邊防火牆 -> 建立防火牆規則
  2. 輸入名稱及圖中欄位(欄位值須相同)
  3. 點選建立便完成

安裝Valheim Server

回到VM執行個體 -> 在列表會看到剛剛建立的vm，點選SSH開啟Command Line視窗 -> 按步驟輸入下面指令

所有指令要加sudo才會以管理員身分執行

  1. 確認網路安裝工具：sudo apt-get install -y git net-tools
  2. 進入opt資料夾：cd /opt
  3. 下載安裝包：sudo git clone https://github.com/Nimdy/Dedicated_Valheim_Server_Script.git
  4. 進入安裝包資料夾：cd Dedicated_Valheim_Server_Script/
  5. 開啟安裝包權限：sudo chmod +x menu.sh
  6. 進入安裝包選單：sudo ./menu.sh
  7. 選擇安裝伺服器：3, 然後選擇1(安裝) 再按y
  8. 建立 Linux 本地密碼：包含一個數字，一個大寫和一個小寫字母
  9. 建立伺服器名稱
  10. 建立世界名稱(若要轉移舊資料，世界名稱須和舊世界相同)
  11. 建立伺服器密碼：至少5個字，包含一個數字，一個大寫和一個小寫字母(連進遊戲時輸入的密碼)
  12. 過程中有需要輸入y/n的都打y
  13. 最後要重啟VM。關閉Command Line視窗，回到VM執行個體 -> 勾選建立的VM -> 點選上方停止按鈕 -> 點選 開始/繼續
  14. 完成後便可開啟遊戲測試連線,在vm執行個體會看到外部IP，這是在遊戲中輸入Join IP時所用到的IP位置，密碼則是步驟11設定的伺服器密碼
  15. 成功後便可開始遊戲，如果之前玩到一半要遷移舊資料到VM的看搬移地圖檔案
伺服器更新

當官方有任何更新時必須手動重啟伺服器，更新內容才會同步到遊戲中，停止後須等待約1分鐘再啟動

  1. 停止伺服器：sudo systemctl stop valheimserver.service
  2. 啟動伺服器：sudo systemctl start valheimserver.service

或是到上面步驟6的選單中輸入代號重啟14

若重啟後有任何問題可以停止伺服器 -> 重啟VM -> 啟動伺服器

搬移地圖檔案

  • 要搬移Valheim地圖資料須將worlds/資料夾下所有檔案搬移過去,不同主機存放目錄不相同
    Windows
    C:\Users\USERNAME\AppData\LocalLow\IronGate\Valheim\worlds\
    Linux
    /home/steam/.config/unity3d/IronGate/Valheim/worlds/
  • 我們須透過SFTP將資料上傳至VM中
取得公私鑰

要連線至VM中最快的方式是使用金鑰的方式

  1. 首先下載PuTTYgen並執行
  2. 勾選下面幾項後按Generate
  3. 修改Key comment為建立VM的mail名稱(例如我的mail是example@gamil.com則輸入example)
  4. 複製圖中紅框部分之後會用到，然後按Save private key(有任何彈窗都按確認)
  5. 回到GCP -> 點選左邊工具列 -> Compute Engine -> 中繼資料 -> 安全殼層金鑰
  6. 點選編輯 -> 新增項目 -> 將複製內容貼到這裡 -> 儲存
  7. 到這裡便完成公私鑰的設定了
上傳地圖檔
  1. 首先下載FileZilla
  2. 開啟後 檔案 -> 站台管理員 -> 新增站台(名稱自訂)
  3. 按下圖輸入欄位，藍框部分須和圖中相同，紅框填入自己的
    • 主機填入VM的對外IP
    • 使用者和Key comment的值相同
    • 金鑰檔案是之前儲存的private key檔案位置
  4. 點選連線，若連線成功代表設定正確，若上面有任一步驟錯誤皆可能讓連線失敗
  5. 將worlds/資料夾下所有檔案先拖移到/home/USERNAME/底下
  6. 開啟VM command line
  7. 停止伺服器：sudo systemctl stop valheimserver.service
  8. cd到USERNMAE目錄(EX:cd /home/example)
  9. 搬移所有檔案到worlds目錄 : sudo mv * /home/steam/.config/unity3d/IronGate/Valheim/worlds/
  10. 按伺服器更新步驟重啟伺服器
  11. 開啟遊戲確認地圖是否已更新
