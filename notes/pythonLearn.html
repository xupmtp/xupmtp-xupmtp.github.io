前言

  • 目前python主流的版本為2.7和3.X,學習推薦從高版本的3.X開始,除非遇到非用python2不可的情形,否則未來使用python2的機會只會越來越少.
  • 學習前可以先想想要用python做什麼,學習的目標會較明確,目前python的主要應用有:
    1. AI/機器學習
    2. 大數據/資料分析
    3. 遊戲設計
    4. 網站開發
    5. 網路爬蟲
    6. 文字/數學處理

有名的線上學習網,不過我沒實際用過,大多要付費但免費的資源也很多,有興趣也可以搜尋其他python相關課程聽聽看

刷題網站

刷題的主要目的是為了面是做準備,上面有許多企業在面試時會出的考古題,不過新手也可以上去練習看看,可以更加熟悉語法

LeetCode

最有名的刷題網,題目齊全且支援的程式語言也很多種,題目有分類型和難易度可依自身需求做調整,老手新手都很實用

Codewars

上面題目多為使用者自創,網站結合遊戲性讓寫題目更有趣,有許多有趣的題目

開發工具

有些教學網站可以線上練習,但最好還是下載一個離線的工具可以隨時使用,推薦順序從上到下

PyCharm

首推!功能很齊全我以前也用這個,有分免費版和付費版,學習上使用免費版就很足夠了

Jupyter

功能強大的開發工具,許多知名企業與大學都使用Jupyter做開發與教學

Spyder

輕量的工具,提供方便查看執行結果的畫面而且它是完全免費的工具

Sublime Text3

很輕量的工具,打開跟執行都很快

Visual Studio Code

微軟開發的編輯工具,如果要同時寫Html,javaScript等等程式它是首選,單純學習python它也是很好用的工具

Notepad++

最多人用的編輯軟體,基本上什麼程式都可以編輯,不過其主要功能為編輯而非開發所以學習的話不推薦使用
